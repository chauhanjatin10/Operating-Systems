Instructions - 

1) Compile using -> g++ dphil-CS17BTECH11019.cpp -o dphil -lpthread -w 

2) Run the correspoding input file using ->

		./dphil < input.txt
		
A log file - dphil-log.txt will be created, containing the details of all access times of the threads.
Another log - Times.txt file will be created containing the details of average waiting time and worst
case waiting time for the threads.
